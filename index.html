Acrònims i Sigles


  1. ADL: Archetype Design Language
  2. AM: Archetype Model -> consists of the archetype language definition language (ADL), the Archetype Object Model (AOM) and the openEHR Archetype profile (oAP)
  3. AOM: Archetype Object Model.
  4. API: Application Programming Interface
  5. BO: bussiness objects.
  6. BP: business partner
  7. CDR: Clínical Data Repository. On es desa la informació clínica d’una persona en format OpenEHR.
  8. CIM: Clinical Information Models.
  9. CKM: Clinical Knowledge Manager
  10. CMBD: conjunt mínim bàsic de dades
  11. CPF: Catàleg de Productes Farmacèutics
  12. ECAP: estació clínica d’atenció primària
  13. Entorn de PRE: parlant d’un software o sistema de informació, entorn de proves, o Preproducció.
  14. Entorn de PRO: parlant d’un software o sistema de informació, entorn real, amb usuaris finals, o de Producció.
  15. ETL: Extract, transform, load.
  16. HC3: Història Clínica Compartida a Catalunya.
  17. HIS: Sistema Informació Hospitalària. Hospital Information System.
  18. HL7-CDA: Clinical Document Architecture. Estàndard d’interoperabilitat en documents clínics informes d’alta, de resultats de proves, etc). Format de la informació a HC3.
  19. ICAM: Institut Català d’Avaluacions Mèdiques (inspecció mèdica de baixes etc)
  20. ICO : Institut Català d’Oncologia
  21. IDI: Institut de Diagnòstic per la Imatge
  22. IPT: informe de posicionament terapèutic
  23. KPI: Key Performance Index
  24. LES: llistes d’espera
  25. MHDA: Medicació Hospitalària de Dispensació Ambulatòria.
  26. MMA: Metodologia del Modelatge d’Arquetips
  27. MPI: Master Patient Index. Guarda la informació demogràfica relacionada amb el pacient però no la clínica.
  28. MVP: mínim producte viable
  29. PO: Product owner
  30. POC: proof of concept
  31. PREM: patient reported experience measures. La impressió dels pacient sobre el sistema de salut que se li aplica
  32. PROM: Patient Reported Outcome Measure. L’estat de salut del pacient en un moment determinat
  33. RCA: Registre Central Assegurats.
  34. RCDS: Registro central de datos de salud. Indicadors de salut.
  35. REVAC: registre de Vacunes
  36. RFI: request for information. Document de petició de informació a proveïdors (empreses privades) sobre un producte existent o a desenvolupar.
  37. RFQ: request for quotation. Document de “licitació” en el que es publica que es demana un producte amb certes característiques i es demanen ofertes de preus a proveïdors.
  38. RPT: Registre de Pacients i Tractaments
  39. RSA: Sistema de generació de registres per part de l’usuari.
  40. SEIS: Sociedad Española de Informática y Salud
  41. SIFARE: sistema d'informació farmacèutica de recepta
  42. SIMDCAT: Sistema d’Imatge Digital Mèdica de Catalunya
  43. SIRE: Sistema Integrat de Recepta Electrònica
  44. SISCAT: Sistema Sanitari Integral d'utilització pública de Catalunya. Tots els centres sanitaris (hospitals Aguts, psiquiàtrics, sociosanitaris, centres A. Primària...) de gestió privada però amb finançament concertat amb el CatSalut (p. exemple hospital Mar, hospital de Vic,...)
  45. XDS - Cross Enterprise Document Sharing, a system for sharing clinical documents between healthcare institutions.
